Aplicación Interactiva de carrito de la compra con base de datos y APIs

Es necesario tener Node instalado, mínimamente los módulos contenidos en el zip "node_modules"
Para ejecutar el servidor: 'node servidor.js'

La página principal de los clientes es 'http://localhost:3001' o 'http://ip_de_tu_ordenador:3001'

La página de los trabajadores es 'http://localhost:3001/trabajadores.html' o 'http://ip_de_tu_ordenador:3001/trabajadores.html'

En la web móvil de los clientes, pulsar el botón de micrófono para operar por voz, está todo explicado con menús.
Si pulsas el botón del megáfono, se leerá en voz alta el carrito .
Si pulsas el botón de las tres barras de arriba a la derecha se deslizará el carrito.
Si agitas el móvil se procede con el pago del carrito.

En la web de los trabajadores van apareciendo los pedidos en tiempo real, y pueden ser marcados como pagados.
